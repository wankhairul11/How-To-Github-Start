How to Setup Git/Github
1. right click working folder, open git bash
2. git config --global user.name"username github"
3. git config --global user.email"email"
4. git init
5. touch index.html
6. git add . (titik tuu add all)
7. git commit -m "commit message cth: file added"
8. git status
9. git push origin main
10. git remote add origin https://github.com/username/username.git

additional command:
git rm (remove file)
ls (list files/folder)
ls -a (show hidden files)
